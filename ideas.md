## Idea 1: "Code Terminal" - Neo-Terminal Aesthetic

**Design Movement:** Terminal/Hacker aesthetic kết hợp với modern glassmorphism

**Core Principles:**
- Dark-first design lấy cảm hứng từ terminal/IDE
- Monospace typography tạo cảm giác "coding environment"
- Neon accent colors (cyan, green, amber) trên nền tối
- Glassmorphism panels với backdrop blur

**Color Philosophy:**
- Background: Deep navy (#0a0e1a) → gợi cảm giác coding lúc đêm khuya
- Primary accent: Electric cyan (#00d4ff) → năng lượng, công nghệ
- Secondary: Emerald green (#00ff88) → success, progress
- Warning: Amber (#ffb800) → attention
- Text: Cool gray tones

**Layout Paradigm:**
- Split-panel layout giống IDE (sidebar + main content + optional right panel)
- Terminal-style header với blinking cursor
- Card-based lesson grid với hover glow effects

**Signature Elements:**
- Typing animation cho tiêu đề
- Terminal-style progress bars
- Code block styling giống VS Code

**Interaction Philosophy:**
- Keyboard-first navigation
- Smooth scroll với parallax subtle
- Hover effects với glow/neon trails

**Animation:**
- Text typing effect cho headings
- Fade-in stagger cho lesson cards
- Progress bar fill animations
- Subtle pulse cho active elements

**Typography System:**
- Display: JetBrains Mono (bold) cho headings
- Body: Inter cho readable content
- Code: Fira Code cho code blocks
</text>

## Idea 2: "Knowledge Atlas" - Editorial Cartography

**Design Movement:** Editorial design meets data visualization, lấy cảm hứng từ bản đồ học tập

**Core Principles:**
- Clean editorial layout với generous whitespace
- Warm neutral palette tạo cảm giác sách giáo khoa cao cấp
- Information hierarchy rõ ràng qua typography scale
- Progressive disclosure - thông tin được reveal dần

**Color Philosophy:**
- Background: Warm off-white (#faf8f5) → giấy cao cấp
- Primary: Deep teal (#0d6e6e) → tri thức, chiều sâu
- Accent: Coral (#e8634a) → điểm nhấn, CTA
- Neutral: Warm grays → editorial feel
- Code blocks: Soft cream background

**Layout Paradigm:**
- Magazine-style asymmetric grid
- Left sidebar navigation dạng "table of contents"
- Main content area với wide margins
- Floating progress indicator

**Signature Elements:**
- Chapter-style lesson numbering với decorative borders
- "Learning path" visualization dạng connected dots
- Pull-quote style cho key concepts

**Interaction Philosophy:**
- Smooth page transitions
- Bookmark/highlight system
- Reading progress indicator

**Animation:**
- Page turn transitions
- Smooth accordion expansions
- Counter animations cho statistics
- Subtle parallax on scroll

**Typography System:**
- Display: Playfair Display cho chapter titles
- Body: Source Sans 3 cho nội dung
- Code: Source Code Pro
</text>

## Idea 3: "Quantum Learn" - Geometric Brutalism meets Soft UI

**Design Movement:** Neo-brutalism nhẹ nhàng kết hợp với geometric patterns

**Core Principles:**
- Bold geometric shapes với rounded corners mềm mại
- High contrast color blocking
- Playful nhưng professional
- Chunky borders và bold shadows

**Color Philosophy:**
- Background: Light cream (#fffef5)
- Primary: Electric blue (#2563eb) → trust, technology
- Secondary: Bright yellow (#fbbf24) → energy, learning
- Accent: Hot pink (#ec4899) → attention
- Borders: Black (#1a1a1a) → definition

**Layout Paradigm:**
- Bento grid layout cho lesson overview
- Stacked card layout cho lesson content
- Floating action buttons
- Offset grid với overlapping elements

**Signature Elements:**
- Thick black borders (3-4px) trên cards
- Bold drop shadows (offset, solid color)
- Geometric decorative elements (circles, triangles)

**Interaction Philosophy:**
- Bouncy hover animations
- Card flip effects
- Drag-like interactions
- Satisfying click feedback

**Animation:**
- Spring physics cho hover states
- Stagger reveal cho grid items
- Rotate/scale micro-interactions
- Confetti on quiz completion

**Typography System:**
- Display: Space Grotesk (bold) cho headings
- Body: DM Sans cho content
- Code: IBM Plex Mono
